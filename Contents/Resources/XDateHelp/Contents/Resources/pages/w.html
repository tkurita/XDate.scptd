XDate Reference
w

w()

Return weekday name in English.

The output format can be customized with set_weekday_format.

Result

text

Example
tell script "XDate"'s make_with("2018/1/1")
log w() -- "Monday"
set_weekday_format("short")
log w() -- "Mon"
set_weekday_format("number")
log w() -- "2"
end tell
