time_to_GMT

time_to_GMT(delim)

Return time to GMT as text.

Parameters
  • delim (text) : the delimiter between hour and minute.
Result

text

Example
tell script "XDate"
time_to_GMT(":") -- "+9:0"
set_leading_zero(true)
time_to_GMT(":") -- "+09:00"
time_to_GMT("") -- "+0900"
end tell
