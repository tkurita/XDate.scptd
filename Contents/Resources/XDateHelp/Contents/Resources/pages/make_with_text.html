make_with_text(a_text)

Make a XDate instance from a text.

Parameters
  • a_text (text) : text can be converted to date class
Result

XDate instance

Example

make_with("2018/1/5 7:08")
-- exivalent to make_with_data(date "2018/1/5 7:08")
end tell
